# Rock-Paper-Scissors
This project about the Rock Paper and Scissors
 Rock Paper Scissors Game
#Project Description

This is a simple Rock Paper Scissors game built using HTML, CSS, and JavaScript.
The game allows a user to play against the computer. The computer makes a random choice, and the winner is decided based on standard game rules.

This project is great for beginners to understand DOM manipulation, events, and basic game logic in JavaScript.

#Technologies Used

HTML – Structure of the game

CSS – Styling and layout

JavaScript – Game logic and interactivity

3Features

User vs Computer gameplay

Random computer choice

Instant result display (Win / Lose / Draw)

Simple and responsive UI

Beginner-friendly code

3How to Play

Open the game in your browser.

Click on Rock, Paper, or Scissors.

The computer will automatically choose one option.

The result will be displayed on the screen.

Play again as many times as you want.
